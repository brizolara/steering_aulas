# steering_aulas
Template para alguns steering behaviours.
Cada behaviour está em um script diferente - no momento, seek_and_flee.cs, Arrive.cs, PathFollow.cs
Experimente:
- Montar uma cena usando Fundo_Thayna.jpg, colocando em Path os waypoints que aparecem na imagem e calibre SteeringManager.cs (ou PathFollow.cs) para que agente percorra o caminho
- Modificar ao vivo os parâmetros de SteeringManager.cs
